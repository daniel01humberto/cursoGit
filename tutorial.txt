TUTORIAL GIT

Iniciando repositório

$ git init

adicionando arquivo no índice

$ git add arquivo.txt

Comitando

$ git comit -m 'mensagem'

Adicionando repositório remoto

$ git remote add origin url

enviando para repositório remoto

$ git push

criando branch

$ git branch develop

Mudando de branch

$ git checkout develop

Realizando merge

$ git merge master

Criando tarefa DESENV-02

Commit na develop
